# Debug

Two options — ask the user which one.

- **Orbit** (`debug-orbit.md`) — observability dashboard: requests, SQL, logs, exceptions, cache, ORM events, emails, background jobs, Redis, outgoing HTTP. Optional MCP server lets AI assistants query live telemetry. Pick for general debugging.
- **Silk** (`debug-silk.md`) — profiling dashboard with `@silk_profile` for specific Python functions. Pick when you need to pinpoint CPU time inside known code paths.

Apply the matching reference; both are dev-only.
